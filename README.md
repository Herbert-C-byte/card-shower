## How to Use

### Getting Started
1. Open `index.html` in your web browser
2. Click "Next Message" or "Random" to see different greetings
3. Click the heart icon to add messages to your favorites
4. Use the buttons to access different features

### Keyboard Shortcuts
- **Space/Enter** - Next message
- **R** - Random message
- **T** - Speak/Pronounce message
- **C** - Copy message
- **F** - Add to favorites
- **H** - Shuffle mode (15 random messages)
- **S** - View statistics
- **Esc** - Close modals

### Features Explained

**Quiz Mode**: Test your language knowledge by guessing which language each greeting belongs to. Get instant feedback on your answers.

**Shuffle Mode**: Watch 15 messages quickly shuffle across the screen. Great for memorization!

**Statistics**: Track:

- Unique languages discovered
